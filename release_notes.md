## ProcIR v1.2.0 - Windows 应急响应进程排查工具

面向安全工程师的一键式应急响应工具，快速定位木马、后门、持久化、白加黑、内存注入等威胁。

### v1.2.0 新增功能

- **中英文双语 UI**：页面右上角一键切换中/英文界面，语言偏好自动保存
- **Claude AI 集成**：英文模式下 AI 分析使用 Anthropic Claude API（Sonnet 4.6 / Haiku 4.5）
- **MiniMax AI 保留**：中文模式下继续使用 MiniMax 大模型（M2.5 / M2.7）
- **英文系统 Prompt**：英文模式使用专业英文应急响应 Prompt
- **双语文档**：README 支持中英文切换

### v1.1.0 功能

- **AI 智能分析**：集成 MiniMax 大模型，支持一键将扫描结果投喂给 AI 进行安全研判
  - 多轮对话，上下文连续追问
  - 完整数据投喂 / 摘要投喂两种模式
  - Token 消耗实时统计

### 核心能力

- **9 大分析引擎**：运行态 / 触发态 / 历史态 / 事件态 / 模块态 / YARA / 内存分析 / IOC 监控 / AI 智能分析
- **13 个分析视图**：活跃进程 / 触发器 / 执行对象 / 历史取证 / 事件日志 / 模块分析 / 时间线 / 行为链 / IOC / YARA / 内存分析 / IOC 监控 / AI 分析
- **多维融合评分**：基础规则 → 强规则 Override → 组合加权 → 白特征抵消 → 上下文权重
- **6 种攻击链识别**：宏攻击链 / 浏览器利用链 / 持久化执行链 / WMI 后门链 / DLL 侧加载链 / 下载执行链

### 特性

- 单文件运行，无需安装
- 纯 Go 实现，无 CGO，无外部依赖
- 非常驻，扫描完即退
- 中英文双语界面，一键切换
- 内嵌 Web UI，自动打开浏览器
- 建议以管理员权限运行

### 使用

```
procir.exe                    # 直接运行
procir.exe -yara rules.yar   # 带 YARA 规则
```
