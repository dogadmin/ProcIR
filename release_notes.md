## ProcIR v1.5.2 - Windows 应急响应进程排查工具

面向安全工程师的一键式应急响应工具，快速定位木马、后门、持久化、白加黑、内存注入等威胁。

### v1.5.2 — Bug 修复 & 检查更新

本次更新修复了安全审计发现的多个问题，新增检查更新功能。

> 因近期代码变动频繁，临时加入了「检查更新」按钮方便用户获取最新版本，后续版本稳定后会考虑移除。

#### 安全修复

- **修复 YARA 上传路径穿越漏洞**：上传文件名未做清理，`../../../evil.yar` 可逃逸临时目录写入任意路径。现使用 `filepath.Base()` 强制提取文件名

#### Bug 修复

- **修复内存分析潜在死循环**：`VirtualQueryEx` 返回 `RegionSize=0` 时循环无法推进，现增加零值检查
- **修复 YARA parseSize 边界 panic**：空字符串或过短输入触发越界切片
- **移除 IOC base64 误报提取**：SHA256 哈希、证书数据等被错误匹配为 base64，已移除该检测

#### 新功能

- **检查更新按钮**：工具栏新增「检查更新」，点击后查询 GitHub Releases 最新版本。有新版本弹窗提示并跳转下载页，无更新则提示已是最新。仅用户主动点击时触发，不会自动联网

---

### 使用

```
procir.exe                    # GUI 模式（默认）
procir.exe -cli -o scan.json # CLI 模式，导出 JSON
```
